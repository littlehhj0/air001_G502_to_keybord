# 基于air001的G502鼠标转键盘
## 环境
可使用arduino ide编译下载
或vstudio中的vMicro编译下载
## 注意事项
下载时时钟选项需选择（HSI 16Mhz, HCLK 16Mhz）
## 介绍
本项目是一个基于air001的G502鼠标转键盘的项目，使用了air001的开发板和G502鼠标，通过编写代码实现了将鼠标的按键转换为键盘输入的功能。# 基于air001的G502鼠标转键盘
